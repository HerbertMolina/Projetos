# 🎯 TryHackMe Write-ups

Bem-vindo à minha coleção de write-ups do TryHackMe.

Este diretório reúne a documentação técnica dos laboratórios que realizo na plataforma TryHackMe como parte do meu processo contínuo de desenvolvimento em Segurança da Informação.

O objetivo não é apenas registrar a solução de cada máquina, mas documentar o raciocínio utilizado durante o reconhecimento, enumeração, exploração e pós-exploração, produzindo relatórios organizados e reproduzíveis.

---

# 🎓 Objetivos

Os laboratórios publicados neste repositório têm como foco:

* Desenvolver habilidades práticas em Cibersegurança.
* Consolidar metodologias utilizadas em Pentest e Blue Team.
* Aprimorar processos de documentação técnica.
* Construir um portfólio baseado em atividades práticas.
* Registrar aprendizados e técnicas utilizadas durante cada desafio.

---

# 🧠 Áreas abordadas

Os write-ups podem envolver temas como:

* Linux
* Windows
* Web Security
* Active Directory
* Privilege Escalation
* OSINT
* Reconnaissance
* Enumeration
* Vulnerability Assessment
* Exploitation
* Lateral Movement
* Post Exploitation
* Digital Investigation

---

# 📄 Estrutura dos Write-ups

Cada máquina segue, sempre que possível, uma estrutura padronizada contendo:

1. Informações da máquina
2. Objetivos
3. Reconhecimento
4. Enumeração
5. Exploração
6. Escalada de Privilégios
7. Flags obtidas
8. Ferramentas utilizadas
9. Evidências
10. Lições aprendidas

---

# 📂 Organização

Cada laboratório possui sua própria pasta contendo:

* README.md
* Relatório técnico
* Evidências
* Capturas de tela (quando aplicável)

---

# 📚 Objetivo do Portfólio

Este material tem finalidade exclusivamente educacional e foi produzido em ambientes autorizados para treinamento e desenvolvimento profissional.

Todo o conteúdo respeita as políticas da plataforma TryHackMe e não deve ser interpretado como incentivo à realização de atividades sem autorização.
